# Anatomia

```css
h1 { /* nome do elemento */
    /* declaração CSS */
    color: blue;  /* propriedade: valor */
    font-size: 60px;
    background: gray;
}
```

* Selector (nesse caso "h1")
* Declaration (o bloco de código de h1)
* Properties (exemplo: "color")
* Property Value (exemplo: "blue")
