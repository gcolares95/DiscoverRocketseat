Comentários

Para fazer um comentario em HTML usamos:

<!--Conteúdo-->

Anatomia de Tags

Anatomia de tags consiste em:

  • Abertura de tags
  • Fechamento de tags
  • Conteúdo
  • Elementos
Obs: as vezes temos tags que não precisam de fechamento, pois elas não tem conteúdo.

<-- Tags de abertura e fechamento -->
<p> Abertura de tag
</p> Fechamento de tag

<-- Conteúdo da tag -->
<h1>Conteúdo da tag</h1>

<-- Elementos Vazios -->
<img src="" alt="">

Atributos

São informações extras e informações sobre a tag. Podem ser valores booleanos que não precisam de conteúdo.

<img src="conteudo" alt="textoAlternativo">

<input type='text' disabled>

As aspas podem ser simples ' ' ou aspas duplas " "

Atributos Globais mais utilizados

<class>
<contenteditable>
<data-*>
<hidden>
<id>
<style>
<tabindex>
<title>

Aninhamento de Tags

Aninhamento de tags consiste em 3 coisas:

  • Hieraquia
  • Fluxo
  • Posicionamento dos elementos

<p>
  Vou <em>escrever um parágrafo</em>
</p>
