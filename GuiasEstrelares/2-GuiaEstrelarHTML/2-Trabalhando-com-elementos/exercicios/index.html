Listas

Nas listas me HTML, temos listas ordenadas e não ordenadas.

Ordenadas:

<ol>
<li>item 1 da lista <li>
<li>item 2 da lista <li>
</ol>

Não ordenadas:

<ul>
<li>item 1 da lista</li>
<li>item 2 da lista</li>
</ul>

Representando códigos

Para representar códigos usamos <code> e <pre>.

Há um diferença entre essas duas tags, <code> é usado para as partes genéricas do código.

Já o <pre> é um bloco de código que vai manter os espaços e os recuos terá o código.

<code>
<h1>Title here</h1>
</code>

<pre>
   <h1>Title here</h1>
</pre>

Elementos genéricos

<div> é utilizada para criar uma divisão ou numa seção em um documento HTML.

<span> é um elemento inline.

Exemplo:

Código HTML

<span>Utilizando o span1</span>
<span>Utilizando o span2</span>

<div>Utilizando o div1</div>
<div>Utilizando o div2</div>
